# Example Next.js app (for demos)

This is a tiny Next.js app you can deploy to Vercel and use as a predictable target for PR QA Copilot.

## Run locally
```bash
cd examples/nextjs
npm install
npm run dev
```

## Deploy to Vercel
- Import this folder as a project.
- Use the Vercel Preview URL as `base_url`.
